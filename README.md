# sao-backstage

> A generator for backstage systems

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao backstage my-project
```

### From git

```bash
sao iuiaoin/sao-backstage my-project
```
